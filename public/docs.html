Installation

No advanced installation is necessary. Simply fire up a server and let your friends join the fun. They will need to know the host address of your server, i.e. the IP-address. If you don't know it, click here to find out.

Server setup

Start the server by double clicking the executable jar file or from the command line by typing the following command in a terminal:

java -jar pictionary-server <port>

Join a game

Start the client by double clicking the executable jar file or from the command line by typing the following command in a terminal:

java -jar pictionary-client.jar

Fill in some connection details and a username

Gameplay

The game will start when four people has connected to the server. The first one to connect will be the first one to draw.

Drawing

Drawing is easy. Just select a pen size using the slider and a color from the color bar and start drawing. Your opponents will be able to see the picture in realtime.

Scoring system

Your opponents has to guess the correct word within sixty seconds, otherwise no points will be rewarded to either the player who guessed or the player drawing. The points will incrementally decrease as time passes and both the player drawing and the player who guessed the correct word will be awarded the same amount of points.
